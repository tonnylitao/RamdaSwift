### Installation
```ruby
pod 'Ramda'
```


## API
### __

```swift
R.add(R.__, 1)(2) // 2 + 1
```

### add

```swift
R.add(2, 3)       // 5
R.add(7)(10)      // 17
```

### addIndex

```swift
R.add(2, 3)       // 5
R.add(7)(10)      // 17
```
